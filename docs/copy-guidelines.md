# docs/copy-guidelines.md

Tone
- Luxury restraint + technical authority
- Short, confident sentences
- No hype, no clichés

Forbidden phrases
- “best-in-class” (unless proven)
- “cutting-edge” (replace with specifics)
- “revolutionary”
- “we one-stop shop”
- “fast and cheap”

Preferred language
- disciplined execution
- inspection-ready installs
- documented closeout
- standardized rollout
- scheduling reliability
- after-hours cutovers

Headline patterns
- “Northeast Infrastructure Deployment Done Right.”
- “Install Clean. Document Correctly. Close Out Fast.”
- “Coverage Built for the Northeast Corridor.”

Microcopy examples
- “Send your scope. We’ll confirm coverage.”
- “After-hours window? We can staff it.”
- “Closeout delivered as a clean PDF package.”

CTA patterns
- Verb + outcome:
  - Request Coverage
  - Request Service
  - Download Capability Statement
  - Schedule a Cutover

FAQ writing rules
- Answer in 2–4 sentences
- Include operational detail (scheduling, testing, documentation)
