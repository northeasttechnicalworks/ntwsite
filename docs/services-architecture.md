# docs/services-architecture.md

Services must be exhaustive, operational, and SEO-friendly.
Each service entry includes:
- Description
- Deliverables
- Typical SLA tiers (suggested)
- Example scopes
- Page slug(s)
- Cross-links (related services)

SLA tier definitions (suggested; customize later)
- Standard: 2–5 business days scheduling lead; response within 1 business day
- Priority: 24–72 hours scheduling lead; response within 4 business hours
- Emergency: best-effort same/next day; response within 1–2 hours (availability dependent)

Core service taxonomy

1) Structured Cabling
Slug: /services/structured-cabling
Subservices + slugs:
- Cat6 installation: /services/cat6-installation
- Cat6A installation: /services/cat6a-installation
- Fiber optic cabling: /services/fiber-optic-cabling
- MDF buildouts: /services/mdf-buildouts
- IDF buildouts: /services/idf-buildouts
- Patch panels & termination: /services/patch-panel-termination
- Cable remediation/cleanup: /services/cable-remediation
- Cable labeling: /services/cable-labeling
- Cable testing: /services/cable-testing
- Certification (Fluke or equivalent): /services/cable-certification
- Pathway systems (J-hooks, tray, conduit assist): /services/cable-pathways

Description
Install and remediate structured cabling systems that support enterprise connectivity and security endpoints.

Deliverables
- Labeling plan (or adherence to client plan)
- Termination and dressing standard
- Test results (per run)
- As-built updates (where applicable)
- Photo documentation pre/during/post
- Closeout package PDF

Example scopes
- 200 drops Cat6A across office + IDF expansion
- Fiber run IDF->MDF with enclosure and patching
- Remediation of failed rack terminations and labeling

Cross-links
- Network racks/buildouts
- Wi-Fi/AP mounting
- Surveillance cabling
