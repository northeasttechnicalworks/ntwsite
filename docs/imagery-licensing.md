# docs/imagery-licensing.md

Preferred sources (official/manufacturer first)
- Manufacturer press/media portals and official product pages
- Partner-approved imagery

Stock sources (high-quality)
- Getty Images
- Shutterstock (enterprise tier)
- Stocksy (editorial feel)
- Adobe Stock

Forbidden sources
- Unlicensed Google Images
- Watermarked previews
- Random reseller product JPGs

Asset rules
- Maintain a manifest:
  - filename, source URL, license, date acquired, usage pages
- Keep consistent color grading (cool, industrial)
- No cheesy smiling contractor shots

Icons
- Lucide (ISC license) or Phosphor (MIT)
- Use consistent stroke width across site

Diagram assets
- Prefer custom SVG authored for NTW (in-house)
- Keep a /assets/diagrams folder
