# docs/graphic-system.md

Graphic intent: Technical elegance. Blueprint-like diagrams + industrial luxury materials.

Diagram style (SVG)
- Stroke: 1.25–1.75px (desktop), 1.0–1.25px (mobile)
- Node radius: 2–4px
- Glow: use subtle outer shadow; avoid neon
- Colors:
  - lines: rgba(191,197,206,.28)
  - active lines: rgba(42,106,161,.75)
  - nodes: rgba(230,233,238,.72)
  - active node: #2a6aa1

Diagram motifs
- Cabling path: patch panel -> cable tray -> IDF -> switch
- Surveillance: camera -> PoE switch -> NVR/VMS -> storage
- Access control: reader -> controller -> panel -> door hardware
- Intrusion: sensors -> panel -> monitoring interface
- Network racks: rack elevation + port mapping concept
- Data pulses: moving dot or segment along line (3–6s interval)

Coverage map style
- Custom SVG map (Northeast states)
- Primary: brighter fill with subtle gradient
- Secondary: low contrast
- Boundaries: clean, minimal
- Legend required

Photography style
Use:
- racks, patch panels, cable management
- access control hardware closeups
- cameras in industrial/commercial settings
Avoid:
- smiling contractor stock photos
- cartoon icons
- overly bright, playful gradients

Texture system
- Subtle noise overlay (2–4% opacity)
- Technical grid overlay (very faint)
- “Metal edge” highlight on key panels:
  - 1px top border brighter than bottom

Icon system
Preferred: stroke icons (Lucide/Phosphor) with consistent stroke width.
No mixed icon styles.

Brand mark usage (if available)
- Monochrome white/metal on dark
- Clearspace: 1x logo height
- Do not add gradients to logo

Illustrations guidance
No illustrated characters.
Only schematic or technical diagrams.
